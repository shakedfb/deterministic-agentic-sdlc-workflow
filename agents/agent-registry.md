# agent registry

Welcome to your AI agent development team architecture. This catalog documents agent roles, their interactions, and evolution.

## SDLC Phase Overviews

Browse agents organized by software development lifecycle phase:

- [[requirements-phase]] -- Agents that gather and structure requirements
- [[design-phase]] -- Agents that create architectural and design artifacts
- [[development-phase]] -- Agents that generate and review code
- [[testing-phase]] -- Agents that create and execute tests
- [[deployment-phase]] -- Agents that handle deployment and release
- [[operations-phase]] -- Agents that monitor and maintain systems

## Getting Started

1. Create agent profiles in `design-ideas/` first (low friction capture)
2. Move to `agents/` when ready to formalize (use template for complete schema)
3. Map interactions between agents via wiki links
4. Track metrics and iterate based on testing
5. Version snapshot when making major changes
